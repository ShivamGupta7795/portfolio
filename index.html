An Android Based Distributed System Application

Technologies: Android, Java, XML, Sockets, Quorum Replication, FIFO-Total Ordering

Duration: Feb 2017 - March 2017

Description: The Objective of the project was to implement an Android Based Distributed System which simulates the Amazon Dynamo application. The android AVD's were used to store and share information, where the data gets stored in a specific partition, according to their hash codes. I used Quorum Replication for storing the replicas of the data in the network nodes. These replicas are then used for failure handling and recovery. If a machine fails then it's data can be fetched from it's replicas and during recovery, the machine can fetch previous and other missing data using replicas.

Image Segmentation and Shape Recognition

This project aimed to detect image object shapes. I used segmentation, hough circle transform algorithm, edge detection, gaussian filters and region divide and merge. Hough Transform is a technique to detect shapes of objects whose boundaries are not clear defined i.e. somewhat occluded. Instead of using other edge detection techniques such as Sobel or Canny which will produce noisy edges, the advantage of using Hough transform technique was that it was tolerant of gaps in feature boundary descriptions and is relatively unaffected by image noise. Certain image processing techniques that I used during this project are:
● Gaussian Blur
● Edge Detection
● Thresholding and thinning
● Hough Transform Algorithm

Dimesionality Reduction and Rule Generation

×

Dimensionality Reduction Using PCA, SVD and t-SNE. Frequent Itemset Generation Using Apriori Algorithm. Rule Generation Using Association Algorithm.

Technologies Used: Python3, Numpy, Matplotlib, Jupyter Notebook

Duration: Aug 2017 - Sep 2017

The aim of this project is to perform dimensionality reduction of the given data set by using three methods viz., Principal Component Analysis, Singular Value Decomposition and t- Distributed Stochastic Neighbor Embedding. In some datasets, there are too many attributes which results in a lot of noise and ambiguities. This makes it difficult to project datasets and make predictions and analysis. Dimensionality reduction helps in better representation of data, by reducing the number of variables in consideration, which helps in performing better data analysis operations.
The second part of project aims to generate sets of data items which has support above a given minimum value. We achieve this by using Apriori algorithm. Thereafter, I generated association rules from the given the frequent item sets. The rules are generated according to a given query.
